[Before Installation Information]

Before proceeding with the installation, please ensure that your system meets the following requirements:

- Operating System: Window 8 | Window 10 | Window 11
- Disk Space: 30MB
- Memory: 2GB RAM

Installation Instructions:
1. Double-click on the setup file to start the installation process.
2. Follow the on-screen instructions to complete the installation.
3. If prompted, select the installation directory and other preferences as required.
4. Once the installation is complete, you can launch the application from the desktop shortcut or the start menu.

For any assistance or inquiries regarding the installation process, please refer to the Installation Guide or contact our support team at info@XPACEtechnologies.com
